Projects

symbolic-uaf-detection: Built for my bachelor's thesis, this is a symbolic execution tool using angr to detect Use-After-Free vulnerabilities in binaries. It includes multiple execution modes to mitigate path explosion.
ipc: A FACT and Ghidra plugin built during my time at Fraunhofer for reverse-engineering inter-process communication via static analysis. It detects constructs like os.system, exec* calls, shared memory, named pipes, and message queues - useful for understanding how binaries communicate.
